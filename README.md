# AI 生图与 Prompt 优化工具

这是一个内部使用的网页工具，包含两个功能：

- **AI 生图工作台**：填写 Prompt、选择模型和图片比例后生成图片。
- **Prompt 优化器**：根据文字和参考图，生成更规范的生图 Prompt。

## 打开方式

本地运行后访问：

```text
http://127.0.0.1:3000/
```

进入后会看到“功能大厅”，点击对应按钮进入功能页面。

## AI 生图工作台

1. 第一次进入时输入使用者名称。
2. 可上传一张或多张参考图，也可以不上传。
3. 输入想生成的画面描述。
4. 如果不知道如何组织 Prompt，可点击 Prompt 输入框下方的“`不知道怎么描述prompt？点击跳转进行优化`”按钮。
5. 点击后会跳转到 Prompt 优化器，并自动带上当前已添加的参考图，保持原有顺序；如果已经输入了文字描述，也会一起带过去。
6. 回到生图工作台后，可直接使用优化后的 Prompt 继续生成。
7. 选择生图模型、图片比例和生成张数。
8. 点击“生成图片”。
9. 生成完成后可单张下载，也可全部下载。

右侧“本机记录”只显示当前电脑生成过的记录。

## Prompt 优化器

支持三种任务模式：

- **改图**：保留原图主体与构图，只修改指定部分。输出内容：**1 份 JSON 结构化改图提示词**。
- **参考图生图**：只参考风格、材质、构图和氛围，重新生成新图。输出内容：**1 份自然语言 Prompt + 1 份 JSON 结构化 Prompt**。
- **纯文生图**：不上传图片，只根据文字扩写生成生图描述。输出内容：**1 份自然语言 Prompt**。

使用方法：

1. 选择任务模式，并查看卡片中的“输出内容”说明。
2. 根据需要上传参考图。
3. 填写需求描述、重点保留 / 重点参考、负向约束和风格定位。
4. 点击“调用模型优化 Prompt”。
5. 生成结果会显示在输出文本框中。
6. 自然语言 Prompt 可直接点击“复制”。
7. 如果当前模式支持 JSON 输出，还可以点击“查看 JSON”或“复制 JSON”。
8. 如果需要继续生图，可点击“跳转到生图”，把当前 Prompt 和参考图一起带回生图工作台。

补充说明：

- 从生图工作台点击“**不知道怎么描述prompt？点击跳转进行优化**”进入优化器时，当前参考图和已输入的描述会自动带入，并保持参考图顺序。
- 从生图工作台跳转到优化器后，可以继续补充需求，再输出更适合生图使用的 Prompt。
- 优化器本机记录区域支持鼠标滚轮滚动查看。

## 跨功能使用体验

- 使用者名称在一个功能里登记后，切换到其他功能会自动复用，不需要重复输入。
- 管理员登录后，在当前网页会话内切换功能时不需要重复登录；刷新网页后需要重新登录。

## 管理员功能

在生图工作台右上角点击齿轮进入管理员界面。

管理员可以：

- 管理生图 API 和模型。
- 查看所有生成记录。
- 修改管理员账号和密码。

初始账号：

```text
账号：admin
密码：1596357
```

首次正式使用建议修改管理员密码。

## 注意事项

- 上传参考图越清晰，生成结果越稳定。
- Prompt 尽量写清楚主体、风格、构图、光影、材质和不要出现的内容。
- 生成失败通常与 API 配置、模型限制、网络连接或图片尺寸有关。
- 不建议把 API Key 发给无关人员。
